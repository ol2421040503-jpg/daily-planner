## 使用方法

### 截图功能
1. 点击 **📷 截图(Ctrl+B)** 按钮，或按 **Ctrl+B**
2. 屏幕显示截图选择界面
3. **按住鼠标拖动选择截图区域**（选中区域清晰，未选中区域半透明）
4. 松开鼠标，点击 **"确定"** 保存
5. 按 **ESC** 键可取消截图
6. 截图自动复制到剪贴板，可粘贴到其他应用

### 上传图片
1. 点击 **🖼️ 上传图片** 按钮
2. 选择图片文件
3. 图片自动压缩并显示在文本框下方
